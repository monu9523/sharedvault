# 🔐 SharedVault – Password Manager Web App

A simple and secure password manager web application that allows users to store, manage, and access their credentials efficiently.

---

## 🌟 Features

- 🔑 Add and manage passwords
- 📂 Organized credential storage
- 💻 Clean and responsive UI
- ⚡ Fast and lightweight
- 📱 Progressive Web App (PWA) support
- 🔒 Basic security-focused structure

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js
- Other: Service Workers (PWA), Docker

---
